# IgnisStream Mobile Assets

This directory contains the visual assets for the IgnisStream mobile app.

## Required Assets

To complete the mobile app setup, you'll need to add the following assets:

### App Icons
- `icon.png` - 1024x1024px app icon
- `adaptive-icon.png` - 1024x1024px adaptive icon (Android)
- `favicon.png` - 32x32px favicon for web builds

### Splash Screen
- `splash.png` - 1284x2778px splash screen image with IgnisStream branding

### Default Assets
- `default-avatar.png` - 200x200px default user avatar
- `placeholder-image.jpg` - Generic image placeholder
- `logo-text.png` - IgnisStream text logo
- `logo-icon.png` - IgnisStream icon only

## Design Guidelines

### Color Palette
- Primary Purple: `#7c3aed`
- Secondary Pink: `#ec4899`
- Background Dark: `#111827`
- Background Light: `#ffffff`
- Text Dark: `#1f2937`
- Text Light: `#f9fafb`

### Typography
- Primary Font: System font (SF Pro on iOS, Roboto on Android)
- Gaming Accent: Orbitron (for branding and gaming elements)

### Icon Style
- Use Ionicons from Expo Vector Icons
- Consistent 20px-24px sizing for navigation
- Gaming-themed icons where appropriate

## Asset Generation

You can generate the required assets using tools like:
- **Figma** - For designing custom icons and graphics
- **Icon Kitchen** - For generating app icons in all required sizes
- **App Icon Generator** - Online tools for creating adaptive icons
- **Canva** - For creating splash screens and promotional graphics

## Branding

All assets should maintain the IgnisStream brand identity:
- Gaming-focused aesthetic
- Purple/pink gradient themes
- Modern, clean design
- High contrast for accessibility
- Consistent with web app branding

## Notes

- All images should be optimized for mobile (use WebP when possible)
- Provide @2x and @3x versions for high-DPI screens
- Test icons on both light and dark backgrounds
- Ensure splash screen looks good on various screen sizes
